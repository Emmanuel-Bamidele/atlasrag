Basic usage
Use the copied service token in your app, SDK, or environment. The project below is your default tenant/workspace.
pip install supavector

SUPAVECTOR_BASE_URL=https://YOUR_SUPAVECTOR_BASE_URL
SUPAVECTOR_API_KEY=<paste-the-copied-service-token>

from supavector import Client

client = Client(base_url="https://YOUR_SUPAVECTOR_BASE_URL", api_key="<paste-the-copied-service-token>")
result = client.search(query="hello", collection="default")
