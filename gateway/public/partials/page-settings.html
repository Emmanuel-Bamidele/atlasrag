Billing

Billing and credit

Hosted AI generation and hosted storage are billed differently. Use Dashboard for balances, payment method, invoices, and add-credit flows.

AI credit
Hosted generation uses credit unless the request supplies the matching provider key for the effective provider.
  • ask, code, and boolean_ask can use hosted credit.
  • If you send your own provider key on a supported request, generation can bill against your provider instead of hosted credit.
  • Manage add-credit and current balance in Dashboard.
Storage billing
Hosted storage is separate from AI credit and is tied to retained data over time.
  • Document writes, memory writes, and retained hosted data affect storage usage.
  • Storage is billed separately from AI credit.
  • Use Dashboard for payment method, invoices, and current hosted billing state.
