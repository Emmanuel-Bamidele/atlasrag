Gateway documentation

Technical usage guide for the AtlasRAG semantic gateway, including setup, API usage, and operations.

Architecture

The gateway is a Node.js service that exposes HTTP endpoints for indexing and retrieval. It calls OpenAI to generate embeddings and answers, stores vectors in the C++ TCP service, and stores chunk text in Postgres for previews and citations.

Quick start

  1. Start services: docker compose up -d
  2. Open the UI: http://localhost:3000
  3. Go to Settings and save your API_KEY.
  4. Index a document, then Search or Ask.

Authentication

Protected endpoints require the header x-api-key to match the server's API_KEY environment variable. The UI stores the key in localStorage only.

curl -H "x-api-key: YOUR_KEY" http://localhost:3000/stats

Indexing workflow

Indexing splits text into chunks, embeds each chunk, stores vectors in the C++ TCP service, and stores chunk text in Postgres. Doc IDs must use only letters, numbers, dot, dash, or underscore.

  • Use POST /docs to index raw text.
  • Use POST /docs/url to fetch and index a URL.
POST /docs
{
  "docId": "swe_notes",
  "text": "Your document text..."
}
POST /docs/url
{
  "docId": "swe_docs",
  "url": "https://example.com/article"
}

Search

Search embeds your query and returns the Top K most similar chunks by cosine similarity. Higher K increases recall but can reduce precision and add latency.

GET /search?q=write-ahead%20logging&k=5

Ask (RAG)

Ask retrieves the Top K chunks, then generates a response using only those sources. The response is concise and avoids speculation.

POST /ask
{
  "question": "Explain WAL and why it matters",
  "k": 5
}

Limits and defaults

  • Document size cap: 200,000 chars (server truncates).
  • URL fetch cap: 1,000,000 chars (server truncates).
  • Client file cap: 5 MB.
  • Embeddings are batched; chunk size is capped for embedding safety.

Operational tips

  • Use concise doc IDs; avoid spaces to prevent TCP parsing errors.
  • Lower Top K if responses are slow or noisy.
  • Monitor Metrics for vector and command throughput.

Troubleshooting

  • 401 Unauthorized: ensure API_KEY matches the header.
  • TCP timeout (VSET): ensure doc ID has no spaces, reduce load, or disable vector WAL.
  • Context length errors: re-index with smaller chunks or lower Top K.
