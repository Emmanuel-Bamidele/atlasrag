# Support

SupaVector is currently published as self-hosted software.

## What This Repository Provides

- source code
- local/self-hosted deployment path
- docs for setup and usage
- issue tracking for bugs and feature requests

## What This Repository Does Not Provide

- managed hosting on your behalf
- guaranteed response times
- formal SLA-backed support
- production operations for user deployments

## How To Get Help

For normal product questions or bugs:

- open an issue in the repository with reproduction details

For feature requests:

- open an issue describing the use case and expected behavior

For security concerns:

- do not file a public issue first
- follow `SECURITY.md`

## Good Issue Reports

Include:

- what you tried
- expected behavior
- actual behavior
- logs or error output
- environment details
- whether you are using `docker-compose.yml` or `docker-compose.prod.yml`

## Self-Hosted Expectation

You are responsible for your own deployment, data, backups, secrets, and runtime operations.
